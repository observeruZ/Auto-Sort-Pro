# Auto Sort Pro

![App Preview](Screenshot%202026-06-24%20132032.png)

**Auto Sort Pro** is a simple yet powerful desktop application that helps you organize your messy files automatically with just one click.

---

## ✨ Features

- **Smart File Sorting** – Automatically categorizes files by type
- **One-Click Auto Scan** – Instantly organize your Downloads folder
- **Unsort / Restore** – Easily undo the sorting
- **Beautiful Dark UI** – Clean and modern interface
- **Progress Tracking** – Real-time progress bar
- **Portable** – No installation needed (`.exe` available)

---

## 📂 Supported Categories

| Category     | File Types                          |
|--------------|-------------------------------------|
| Photos       | jpg, jpeg, png, gif                 |
| Music        | mp3                                 |
| Videos       | mp4, avi, mkv, mov                  |
| Documents    | pdf, doc, docx, txt, xlsx, csv      |
| Archives     | zip, rar, 7z                        |
| Others       | Everything else                     |

---

## 🚀 How to Use

1. Download `AutoSortPro.exe`
2. Double-click to run the program
3. Choose one of the options:
   - **Select Folder to Sort** – Choose any folder
   - **Auto Scan Downloads** – Quickly organize your Downloads folder
   - Use **Unsort** buttons to restore files anytime
